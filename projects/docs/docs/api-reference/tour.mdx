# `Tour`

`Tour` is the main entry point for Siteguide.js. Use it to configure a tour, add steps, control navigation, and listen for lifecycle events.

## Export

```ts
class Tour
```

## Constructor

```ts
new Tour(config: TourConfig)
```

Pass a `TourConfig` object when you create the tour. You can start with an empty object and add options over time.

## Common properties

| Property | Type | What it gives you |
| --- | --- | --- |
| `stepList` | `readonly TourStep[]` | The steps currently registered in the tour. |
| `config` | `TourConfig` | The current tour settings, including default values. |
| `activeStep` | `TourStep \| null` | The step that is currently open. |
| `activeStepIndex` | `number \| null` | The zero-based position of the current step. |

## Methods

| Method | Signature | Purpose |
| --- | --- | --- |
| `addStep` | `(config: TourStepConfig) => void` | Adds one step to the tour. |
| `addSteps` | `(steps: TourStepConfig[]) => void` | Adds multiple steps. |
| `removeStep` | `(index: number) => void` | Removes the step with that index. |
| `start` | `() => void` | Starts the tour. |
| `complete` | `() => void` | Finishes the tour and fires the `complete` event. |
| `close` | `() => void` | Closes the tour and fires the `close` event. |
| `prev` | `() => Promise<void>` | Moves to the previous step. |
| `next` | `() => Promise<void>` | Moves to the next step. |
| `goTo` | `(index: number) => Promise<void>` | Jumps to a zero-based step position. |
| `setConfig` | `(config: TourConfig) => void` | Updates the tour settings after creation. |
| `on` | `(type: TourEventType, callback: (tour: Tour) => void) => void` | Subscribes to tour events. |
| `off` | `(type: TourEventType, callback?: (tour: Tour) => void) => void` | Removes one listener or all listeners for a type. |

## When to use it

Use `Tour` when you want one object to own the full flow:

- global behavior such as keyboard control, progress, and close rules;
- the list of steps;
- programmatic navigation;
- lifecycle events such as `start`, `complete`, and `changeStep`.

## Minimal example

```js
import { Tour } from 'siteguide.js';

const tour = new Tour({
    keyboardControl: true,
});

tour.addStep({
    host: '.welcome',
    popup: {
        type: 'text',
        title: 'Welcome',
        text: 'Start here.',
    },
});

tour.start();
```
