# Siteguide.js

Siteguide.js helps you build onboarding tours and guided product walkthroughs with plain JavaScript and CSS. It fits comfortably into framework and non-framework apps because it focuses on the tour itself instead of shipping a full UI system.

## What you get

<div className="sg-inline-list">
  <div className="sg-doc-card"><strong>Flexible targeting</strong> with selectors, DOM elements, or lazy target lookup.</div>
  <div className="sg-doc-card"><strong>Two popup styles</strong>: simple text content or your own custom DOM content.</div>
  <div className="sg-doc-card"><strong>Clear controls</strong> for starting, moving through, and finishing a tour.</div>

## What most projects use

In most projects, you only need:

- `Tour` to create and control a tour.
- `TourConfig` to define tour-wide behavior.
- `TourStepConfig` to describe each step.
- Popup, button, and event types when you want stronger TypeScript guidance.

The package also includes a few secondary exports for advanced use cases. They are listed in the appendix instead of the main documentation flow.
