# Installation

Install Siteguide.js from npm:

```bash
npm install siteguide.js
```

Then load the default stylesheet once in your application entry file:

```js
import 'siteguide.js/dist/css/siteguide.css';
```

## Requirements

- A browser environment with DOM access.
- A bundler or asset pipeline that can import npm packages.
- Modern browser support for common DOM APIs.

## Minimal setup

```js
import 'siteguide.js/dist/css/siteguide.css';
import { Tour } from 'siteguide.js';

const tour = new Tour({});
```

## Without the default CSS

You can skip the bundled stylesheet and write your own styles instead. When you do that, set `classPrefix` in `TourConfig` so your CSS can target the generated markup with predictable class names.

```js
const tour = new Tour({
    classPrefix: 'my-guide',
});
```
