# Installation

You can install Siteguide.js using npm.

<Tabs>
    <TabItem value="npm" label="Npm">
        ```bash
        npm install siteguide.js --save
        ```
    </TabItem>
</Tabs>

## Styles

Import the stylesheet in your project.

```css
import 'siteguide.js/dist/css/siteguide.css';
```
